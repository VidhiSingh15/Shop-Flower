# 🌸 Flower Shop Website

A fully responsive and visually appealing flower shop website built using **HTML**, **CSS**, and **JavaScript**. This project simulates an online florist store showcasing products, allowing users to add items to a wishlist, and providing shop information.

---

## ✨ Features

- 💐 **Featured Products** — Displays a beautiful collection of flower bouquets.
- ❤️ **Wishlist Functionality** — Users can add/remove items using a heart icon. The wishlist appears on the shop page.
- 🛒 **Shop Now Buttons** — Prompts users to take action directly from product cards.
- 🖼️ **Responsive Design** — Optimized for desktop and mobile devices.
- 📋 **About Us Section** — Brief description of the shop's vision and services.
- 📞 **Contact Section** — With business hours and support details.
